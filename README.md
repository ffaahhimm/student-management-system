## 🚀 Quick Start
```bash
git clone https://github.com/ffaahhimm/student-management-system.git
cd student-management-system
python3 student_management.py
```

> No external libraries needed. Just Python 3.

---

## 🎯 Features at a Glance

| | Feature | Description |
|---|---|---|
| 🎨 | **Color-Coded Grades** | Green = pass, Yellow = average, Red = fail |
| 📊 | **Visual Bar Chart** | See every student's marks as ASCII bars |
| 🏆 | **Live Leaderboard** | Full ranking with 🥇🥈🥉 medals |
| 📈 | **Progress Bars** | inline for every student |
| 🎖️ | **Achievement Badges** | Auto-assigned based on performance |
| ↩️ | **Undo** | Instantly undo last add or delete |
| 📦 | **Auto Backup** | Timestamped backups on every exit |
| 📤 | **Export** | One-click CSV or JSON export |
| 📖 | **Subject Marks** | Track individual subject scores |
| 📄 | **Pagination** | Navigate large records page by page |
